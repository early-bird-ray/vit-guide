# ViT KMS (Knowledge Management System)

ViT(Vision Transformer) 관련 자료를 체계적으로 정리한 지식 관리 시스템입니다.

## 폴더 구조

```
KMS/
├── INDEX.md              ← 전체 인덱스 & 키워드 검색 (여기서 시작)
├── README.md             ← 이 파일
├── 01_papers/            ← 핵심 논문 요약
│   ├── vit_original.md       P001: 원 논문 (2020)
│   ├── scaling_vit.md        P002: 스케일링 연구 (2022)
│   └── vit_survey.md         P003: 파생모델 서베이 (2022)
├── 02_huggingface/       ← HuggingFace 자료
│   ├── vit_docs_ko.md        H001: 공식 한국어 문서
│   ├── vit_base_patch16_224.md  H002: 모델 카드
│   └── finetuning_guide.md   H003: Fine-tuning 가이드
├── 03_tutorials/         ← 블로그·강의 자료
│   └── korean_resources.md   T001: 한국어 자료 모음
└── 04_scholar/           ← 학술 레퍼런스
    └── scholar_references.md S001: BibTeX / APA 인용
```

## 사용법

1. **처음 탐색** → `INDEX.md` 열어서 키워드 검색
2. **논문 먼저** → `01_papers/` 폴더
3. **코드 바로** → `02_huggingface/vit_base_patch16_224.md`
4. **한국어 자료** → `03_tutorials/korean_resources.md`
5. **논문 인용** → `04_scholar/scholar_references.md`
