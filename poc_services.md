## 🟢 난이도 낮음

### POC 1. 제품 불량 검출기

> 공장 라인 촬영 이미지에서 스크래치·균열·오염을 자동 분류

| 항목 | 내용 |
|------|------|
| 모델 | `google/vit-base-patch16-224` fine-tuning |
| 데이터 | 정상품 / 불량품 이미지 각 200~500장 |
| 출력 | 불량 여부 + 불량 유형 (스크래치/변형/오염) |
| 스택 | Python + FastAPI + Gradio UI |

**왜 ViT?**  
패턴이 이미지 어느 위치에 있든 Self-Attention이 전역 탐지하므로 CNN보다 위치 불변 결함 탐지에 유리.

---

### POC 2. 음식 칼로리 추정기

> 음식 사진 1장 → 음식 이름 + 예상 칼로리 반환

| 항목 | 내용 |
|------|------|
| 모델 | Food-101 데이터셋으로 fine-tuning된 ViT |
| 출력 | "김치찌개 (약 320kcal)" |
| 확장 | 하루 식단 기록 앱으로 성장 가능 |
| 스택 | HuggingFace + Streamlit + 칼로리 DB 연동 |

---

### POC 3. 반려동물 품종 식별기

> 강아지·고양이 사진 → 품종 + 특성 정보 카드

| 항목 | 내용 |
|------|------|
| 모델 | Oxford-IIIT Pet Dataset fine-tuning (37종) |
| 출력 | 품종명, 성격, 평균 수명, 관련 상품 추천 |
| 비즈니스 | 펫샵·동물병원 앱 연동 |
| 스택 | ViT + FastAPI + React 카드 UI |

---

### POC 4. 피부 상태 분석기

> 셀카 사진 → 건성/지성/복합성 + 트러블 부위 감지

| 항목 | 내용 |
|------|------|
| 모델 | 피부 상태 분류 데이터셋 fine-tuning |
| 출력 | 피부 타입 + 관리 팁 + 추천 성분 |
| 주의 | 의료 진단 아님을 명시 (법적 이슈) |
| 스택 | ViT + 모바일 카메라 API |

## 🟡 난이도 중간

### POC 5. 쇼핑몰 유사 상품 검색

> 상품 사진 찍으면 → 유사한 상품 리스트 반환

| 항목 | 내용 |
|------|------|
| 원리 | ViT의 CLS 벡터 = 이미지 임베딩 → 벡터 DB 유사도 검색 |
| 모델 | `ViTModel` (분류 헤드 제거, 임베딩 추출) |
| 스택 | ViT + FAISS (벡터 검색) + 상품 DB |

**왜 ViT?**  
CNN보다 의미적 유사성 임베딩 품질이 높아 "비슷하게 생긴" 상품을 더 잘 찾음.

```
사용자 사진 → ViT 임베딩 → FAISS 검색 → 유사 상품 Top-5
```

### POC 6. 건물 외벽 균열 자동 점검

> 드론 촬영 이미지 → 균열 위치·심각도 자동 분류 리포트

| 항목 | 내용 |
|------|------|
| 모델 | ViT fine-tuning + Grad-CAM으로 균열 위치 시각화 |
| 출력 | 균열 없음 / 경미 / 보수 필요 + 히트맵 이미지 |
| 시장 | 건설사, 안전점검 대행업체 |
| 스택 | ViT + OpenCV + PDF 리포트 자동 생성 |

---

### POC 7. 문서·영수증 자동 분류기

> 스캔된 문서 이미지 → 종류 분류 + 핵심 정보 추출

| 항목 | 내용 |
|------|------|
| 모델 | ViT로 문서 타입 분류 (영수증/계약서/청구서/명함) |
| 출력 | 문서 유형 + OCR 연동으로 날짜·금액·발행처 추출 |
| 비즈니스 | 회계팀 자동화, 경비 처리 시스템 |
| 스택 | ViT + Tesseract OCR + Excel 자동 정리 |
